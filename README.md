# ProyectoAgenda
Proyecto final del curso de Introduccion a Python

El proyecto es una aplicación de gestión de contactos desarrollada en Python en base al curso de induccion a Python. Permite a los usuarios realizar las siguientes acciones:

Agregar un nuevo contacto: Los usuarios pueden ingresar información como nombre, número de teléfono y categoría para crear un nuevo contacto. La información del contacto se almacena en un archivo de texto.

Editar un contacto existente: Si un contacto ya existe, los usuarios pueden editarlo proporcionando una nueva información, como nombre, número de teléfono o categoría. La aplicación actualiza los detalles del contacto en el archivo correspondiente.

Ver la lista de contactos: Los usuarios pueden ver una lista de todos los contactos almacenados en la aplicación. La información se muestra de manera organizada, lo que facilita la búsqueda de contactos específicos.

Buscar un contacto por nombre: La aplicación permite a los usuarios buscar un contacto específico ingresando su nombre. Si se encuentra un contacto con ese nombre, se muestra su información.

Eliminar un contacto: Los usuarios pueden eliminar un contacto existente proporcionando su nombre. La aplicación borra el archivo de contacto correspondiente.

El proyecto es una herramienta útil para organizar y mantener una lista de contactos personales. Utiliza archivos de texto para almacenar la información, lo que lo hace fácil de usar y comprender. Además, proporciona una interfaz de línea de comandos simple para interactuar con la aplicación.
